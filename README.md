🎂 Birthday Surprise Website

An interactive birthday surprise website built to create a memorable experience for someone special.
The website takes the visitor through a beautiful journey with animations, messages, memories, and a final surprise gift.

This project was created to combine creativity, emotions, and web development into a unique digital birthday celebration.

✨ Features

• 🎈 Balloon pop surprise entry
• 💌 Personal birthday message
• ❤️ Things I love about you section
• 📅 Love memory timeline
• 🤝 Future promises page
• 🎁 Final surprise gift reveal
• 🎆 Confetti and celebration animations
• 📩 Wish message submission (Telegram notification)
• 🎵 Background music support

🛠 Built With

• React.js
• JavaScript
• HTML5
• CSS3
• React Hooks
• Telegram Bot API
